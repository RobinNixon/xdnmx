# Brand Assets To Create

This document lists brand assets that need to be created manually (design work required).

---

## Logo Files Needed

### Primary Logo

Create the xDNMX wordmark in these formats:

| File | Format | Size | Background |
|------|--------|------|------------|
| `logo-dark.svg` | SVG | Vector | For light backgrounds |
| `logo-light.svg` | SVG | Vector | For dark backgrounds |
| `logo-dark.png` | PNG | 512x512 | For light backgrounds |
| `logo-light.png` | PNG | 512x512 | For dark backgrounds |
| `logo-dark@2x.png` | PNG | 1024x1024 | Retina/HiDPI |
| `logo-light@2x.png` | PNG | 1024x1024 | Retina/HiDPI |

### Icon (Favicon)

| File | Format | Size | Use |
|------|--------|------|-----|
| `icon.svg` | SVG | Vector | Scalable icon |
| `favicon.ico` | ICO | 16x16, 32x32, 48x48 | Browser favicon |
| `icon-192.png` | PNG | 192x192 | Android/PWA |
| `icon-512.png` | PNG | 512x512 | Android/PWA |
| `apple-touch-icon.png` | PNG | 180x180 | iOS |

### Social Media

| File | Format | Size | Use |
|------|--------|------|-----|
| `og-image.png` | PNG | 1200x630 | Open Graph (link previews) |
| `twitter-card.png` | PNG | 1200x600 | Twitter cards |
| `github-social.png` | PNG | 1280x640 | GitHub social preview |

## Design Specifications

### Logo Construction

Per the brand guidelines:
- The "x" should be in **JetBrains Mono Bold** (monospace)
- "DNMX" should be in **Inter Bold** or **Space Grotesk Bold**
- Optically align the two typefaces
- Clear space = height of the "x" on all sides

### Color Versions

| Version | x Color | DNMX Color | Background |
|---------|---------|------------|------------|
| Dark on light | Deep Navy #1E293B | Deep Navy #1E293B | Transparent/White |
| Light on dark | White #FFFFFF | White #FFFFFF | Transparent |
| Primary accent | Primary Blue #2563EB | Deep Navy #1E293B | Transparent |

## Recommended Tools

### Free Options

- **Figma** (free tier) — For logo design and exports
- **Inkscape** — Open source SVG editor
- **GIMP** — Open source raster editing
- **Canva** (free tier) — Quick social media assets

### AI-Assisted Options

- **Midjourney** or **DALL-E** — For conceptual exploration
- Note: For text-based logos like xDNMX, manual design typically works better

### Online Converters

- **realfavicongenerator.net** — Generate all favicon sizes from one image
- **squoosh.app** — Optimize PNGs

## Slide Templates Needed

Create templates in:
- Google Slides (most accessible)
- PowerPoint (for corporate environments)
- Figma (for design-savvy organizers)

### Slides to Include

1. **Title slide** — Event name, date, xDNMX branding
2. **Section divider** — For Discovery / Discussion / Building phases
3. **Content slide** — For presenter notes
4. **Code slide** — Dark background, monospace font
5. **Repo showcase** — Screenshot frame + description
6. **Thank you / Next steps** — With xDNMX links

## Priority Order

If limited time/resources, create in this order:

1. **SVG logo** (dark and light versions) — Used everywhere
2. **GitHub social preview** — First impression for repo visitors
3. **Favicon** — Browser credibility
4. **Slide template** — For presenters
5. Everything else as needed
