This folder contains brand guidelines and assets for xDNMX events.

## Contents

- [Brand Guidelines](./brand-guidelines.md) — Colors, typography, usage rules
- `logo/` — Logo files in various formats (coming soon)
- `templates/` — Slide templates, social media templates

---

## Quick Reference

### Colors

| Name | Hex | Usage |
|------|-----|-------|
| Primary Blue | `#2563EB` | Headlines, primary buttons, links |
| Deep Navy | `#1E293B` | Body text, backgrounds |
| Electric Cyan | `#06B6D4` | Accents, highlights, discoveries |
| Warm Orange | `#F97316` | Calls to action, builds, energy |
| Light Gray | `#F1F5F9` | Backgrounds, cards |
| White | `#FFFFFF` | Clean space, contrast |

### Fonts

- **Headlines:** Inter Bold or Space Grotesk Bold
- **Body:** Inter Regular
- **Code:** JetBrains Mono or Fira Code

All fonts are open source and freely available.

### Logo Usage

- Minimum clear space: Height of the "x" on all sides
- Minimum size: 24px height for digital, 0.5" for print
- Don't stretch, rotate, or add effects
- Use on solid backgrounds only
